MQTT Events

Dino is using VerneMQ to broadcast events to clients using MQTT version 5. Below are the possible events a client can receive, and when they are received.

For events, it doesn't matter if they're for a 1-to-1 conversation in a multi-user group, they look the same, but group_type will be different (if specified). Possible values for group_type:

  • 0: multi-user group,
  • 1: 1-to-1.
